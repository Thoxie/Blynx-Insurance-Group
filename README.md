# Blynx Insurance Group

Blynx Insurance Group is a modern insurance advisory platform focused on serving affluent individuals, families, and business owners with complex insurance needs requiring broad market access.

## Getting Started

To run the project locally:

```bash
npm install
```

```bash
npm run dev
```

Then open:

http://localhost:3000

## Project Overview

This project is built using:

- Next.js (App Router)
- TypeScript
- Tailwind CSS

## Repository Structure

- `app/` — application routes and pages
- `lib/` — shared logic and utilities
- `public/` — static assets
- `package.json` — dependencies and scripts

## Deployment

This project is designed to be deployed on Vercel using the GitHub integration.
